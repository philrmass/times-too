# Times Too
### By Phil Mass

A React app for multiplication facts practice, 2s through 12s

Try it [here](https://philrmass.github.io/times-too/)

## To Install
```
git clone git@github.com:philrmass/times-too.git
cd times-too
yarn
```

## To Deploy
```
yarn deploy
```

## To Run Locally
```
yarn start
```
